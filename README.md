## 简述
大学毕业设计作品，系统功能比较少，所有代码个人完成，思路是借鉴网上已有的系统，全部技术是基于JavaScript开发
****
## 所用技术栈
* react
* redux
* react-redux
* react-route4
* antd-mobile
* axios
* express
* socket.io
* mongoose
* mongoDB
## 启动
安装node_modules: npm install   
前端启动： npm start   
后端启动： nodemon server/server.js
